# Sistema de Biblioteca

## Objetivo

Desenvolver um sistema de biblioteca em Java para praticar os conceitos da linguagem, sem utilizar banco de dados ou frameworks.

## Funcionalidades

### Livros
- Cadastrar livro
- Listar livros
- Buscar livro
- Editar livro
- Remover livro

### Usuários
- Cadastrar usuário
- Listar usuários
- Buscar usuário
- Editar usuário
- Remover usuário

### Empréstimos
- Emprestar livro
- Devolver livro
- Listar livros emprestados
- Listar livros disponíveis

## Regras de Negócio

- Um livro só pode ser emprestado se estiver disponível.
- Cada usuário pode emprestar até 3 livros.
- O ISBN deve ser único.
